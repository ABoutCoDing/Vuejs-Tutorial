# Vue개발자 도구 설치 
1. 빌드 설치

 https://github.com/vuejs/vue-devtools
 zip 다운로드 
 $npm install
 $npm run build
 Chrome 설정 > 도구더보기 > 확장 프로그램을 엽니다.
 "개발자모드"를 체크합니다.
 "압축해제된 확장 프로그램 로드" 버튼을 클릭하고 빌드 경로의 shells/chrome를 선택합니다.

2. 크롬 웹스토어
   크롬 웹스토어에서 "vuejs"로 검색한다.
   vue.js devtools를 설치한다.
